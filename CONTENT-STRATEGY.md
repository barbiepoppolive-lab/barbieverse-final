## Content Types & Formats

### Carousels (Highest engagement)
- 7 slides max
- Slide 1: Hook (stop the scroll)
- Slides 2-6: Value content
- Slide 7: CTA (link in bio)
- Use Barbie character on slides 1 & 7

### Reels (Best reach)
- 15-30 seconds
- Hook in first 3 seconds
- Text overlay for silent viewers
- Trending audio when relevant
- End with CTA

### Stories (Best for engagement)
- Daily polls, questions, quizzes
- Quick tips with swipe-up
- Behind-the-scenes clips
- User testimonials

### Static Posts (Brand building)
- Quote cards with Barbie character
- Infographics
- Revenue proof screenshots
- Community highlights

## Barbie Character Usage

### Where to Use Barbie
- Carousel cover slides (slide 1)
- CTA slides (slide 7)
- Profile picture
- Story highlights cover
- Reel thumbnails
- YouTube channel art

### Consistent Elements
- Same face/look across all content
- Pink color palette
- Professional yet approachable
- Streaming setup in background

### Image Generation Rules
- Always use seed 123456789 for consistency
- Use same prompt structure
- Platform-specific sizing:
  - Instagram: 1080x1080 (square) or 1080x1350 (portrait)
  - Stories: 1080x1920
  - YouTube: 1280x720
  - Twitter: 1200x675
